Mailing lists

There are many ways to get help from the Apache Pulsar community. The mailing lists are the primary place where all Pulsar committers are present. Bugs and feature requests can either be discussed on the dev mailing list or by opening an issue on GitHub.

Name Scope      
users@pulsar.incubator.apache.org User-related discussions Subscribe Unsubscribe Archives
dev@pulsar.incubator.apache.org Development-related discussions Subscribe Unsubscribe Archives
commits@pulsar.incubator.apache.org All commits to the Pulsar repository Subscribe Unsubscribe Archives
