Pulsar Perf 性能测试

Pulsar Perf 是 Apache Pulsar 内置的性能测试工具。 你可以用 Pulsar Perf 测试消息写入或读取性能。 更多性能优化细节，请参阅 这里。

生产消息

本示例展示 Pulsar Perf 如何使用默认选项生产消息。 pulsar-perf produce 命令的全部可用配置选项，可参考配置选项。

bin/pulsar-perf produce my-topic

命令执行后，终端上就会输出连续不断的测试数据。

    配置

    默认情况下，Pulsar Perf 使用 conf/client.conf 文件作为默认配置，将 conf/log4j2.yaml 作为 Log4j 默认配置。 如果你想连接到其他 Pulsar 集群，你可以更新客户端配置中的 brokerServiceUrl。

    您可以使用以下命令来更改配置文件和 Log4j 配置文件。

    export PULSAR_CLIENT_CONF=<你的配置文件>
export PULSAR_LOG_CONF=<你的日志配置文件>

    此外，你可以使用以下命令通过环境变量设置 JVM 配置：

    export PULSAR_EXTRA_OPTS='-Xms4g -Xmx4g -XX:MaxDirectMemorySize=4g'

    HdrHistogram Plotter

    HdrHistogram Plotter 是一款查验 Pulsar Perf 测试结果的可视化工具，便于观察。

    可通过如下步骤，实现用 HdrHistogram Plotter 查看相关测试结果：

    1. 从 GitHub 上克隆 HdrHistogram 仓库到本地

      git clone https://github.com/HdrHistogram/HdrHistogram.git

    2. 切换到HdrHistogram文件夹

      cd HdrHistogram

    3. 安装HdrHistogram 绘图器

      mvn clean install -DskipTests

    4. 转换 Pulsar Perf 生成的文件

      ./HistogramLogProcessor -i <pulsar-perf生成的hgrm文件路径> -o <输出文件>

    5. 你将获得两个文件。 将扩展名为 .hgrm 的输出文件上传到HdrHistogram Plotter。
