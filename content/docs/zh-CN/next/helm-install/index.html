使用 Helm 安装 Apache Pulsar

使用官方的 Pulsar Helm chart 在 Kubernetes 上安装 Apache Pulsar。

要求

要在 Kubernetes 上部署 Apache Pulsar，需要满足下面所述要求。

  • kubectl 1.14 or higher, compatible with your cluster (+/- 1 minor release from your cluster)
  • Helm v3 (3.0.2 or higher)
  • Kubernetes 集群，版本 1.14 或以上

环境配置

部署 Pulsar 之前，你需要准备好相关环境。

工具

在电脑上安装 helm 和 kubectl。

云端集群准备

若要创建和连接到 Kubernetes 集群，请参考下面指南：

  • 谷歌 Kubernetes 引擎（GKE）

部署 Pulsar

如果环境设置完成并生成了配置，就可以进行下一步，开始 部署 Pulsar。

升级 Pulsar

若要升级现有的 Kubernetes 安装，请参考 升级文档进行升级。
