使用 Pulsar SQL 查詢資料

在查詢 Pulsar 數據之前，你需要安裝 Pulsar 和內建的連接器。

要求

  1. 安裝 Pulsar。
  2. 安裝 Pulsar built-in connectors。

使用 Pulsar SQL 查詢資料

要使用 Pulsar SQL 查詢數據，請完成以下步驟。

  1. 首先，啟動 Pulsar 本機集群：
./bin/pulsar standalone
  1. 啟動一個 Pulsar SQL worker。
./bin/pulsar sql-worker run
  1. 初始化 Pulsar 本機集群和 SQL worker 後，運行 SQL CLI 。
./bin/pulsar sql
  1. 使用 SQL 命令進行測試。
presto> show catalogs;

---------

(2 rows)

Splits: 19 total, 19 done (100.00%)
0:00 [0 rows, 0B] [0 rows/s, 0B/s]


presto> show schemas in pulsar;

-----------------------

 public/functions      
 sample/standalone/ns1 
(4 rows)

Query 20180829_211818_00005_7qpwh, FINISHED, 1 node
Splits: 19 total, 19 done (100.00%)
0:00 [4 rows, 89B] [21 rows/s, 471B/s]


presto> show tables in pulsar."public/default";

-------
(0 rows)

Query 20180829_211839_00006_7qpwh, FINISHED, 1 node
Splits: 19 total, 19 done (100.00%)
0:00 [0 rows, 0B] [0 rows/s, 0B/s]

由於 Pulsar 中沒有數據，因此不會返回任何資料。

  1. 啟動內建連接器 DataGeneratorSource 並產生一些模擬的數據。
./bin/pulsar-admin sources create --name generator --destinationTopicName generator_test --source-type data-generator

然後，可以在命名空間 "public/default" 中查詢 topic。

presto> show tables in pulsar."public/default";

查詢自己的數據

如果要查詢自己的數據，則需要先導入自己的數據。你可以編寫一個簡單的 producer 並將自定義的數據寫入 Pulsar。以下是一個範例。

public class TestProducer {

    public static class Foo {
        private int field1 = 1;
        private String field2;
        private long field3;

        public Foo() {
        }

        public int getField1() {
            return field1;
        }

        public void setField1(int field1) {
            this.field1 = field1;
        }

        public String getField2() {
            return field2;
        }

        public void setField2(String field2) {
            this.field2 = field2;
        }

        public long getField3() {
            return field3;
        }

        public void setField3(long field3) {
            this.field3 = field3;
        }
    }

    public static void main(String[] args) throws Exception {
        PulsarClient pulsarClient = PulsarClient.builder().serviceUrl("pulsar://localhost:6650").build();
        Producer<Foo> producer = pulsarClient.newProducer(AvroSchema.of(Foo.class)).topic("test_topic").create();

        for (int i = 0; i < 1000; i++) {
            Foo foo = new Foo();
            foo.setField1(i);
            foo.setField2("foo" + i);
            foo.setField3(System.currentTimeMillis());
            producer.newMessage().value(foo).send();
        }
        producer.close();
        pulsarClient.close();
    }
}
